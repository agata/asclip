[ASClip](http://nulab.github.com/asclip/) - ActionScript Clipboard Library
==================================================

これは何？
--------------------------------------

JavaScriptからFlashを使って、システムのクリップボードに文字列をセットできるライブラリです。

使い方
--------------------------------------

examples/以下のHTMLファイルをご覧ください。
